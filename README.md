# ai_portfolio_regression_california_housing

Titre : California Housing Price Prediction using Regularized Linear Regression
Problématique : Prédire le prix médian des maisons en Californie en utilisant les données de recensement.
Méthodologie : EDA, feature engineering, Modèle Ridge/Lasso, Métriques $R^2$/RMSE.
Résultats Clés : meilleur score $R^2$ et  RMSE.
Compétences Démontrées : Pandas, Scikit-learn, Régularisation.
Comment Exécuter : pip install -r requirements.txt.

Un bon modèle doit avoir des résidus distribués de manière aléatoire autour de l'axe y=0. 
Ici on a une forme (un cône), cela signifie que le modèle a du mal à prédire pour certaines plages de prix (clairement les maisons les plus chères).

topic couvert : EDA, baseline, optimisation, évaluation, interprétation, et sérialisation.
